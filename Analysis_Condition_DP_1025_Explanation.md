### 5. 材料定義

#### 5-1. 非線形材料特性 (MATS1)

```nastran
MATS1          1         PLASTIC     10.       4       1      .2     30.
```

**カード解説**:

| フィールド | 値 | パラメータ | 意味 |
|-----------|-----|-----------|------|
| Field 1 | MATS1 | - | 材料応力依存カード |
| Field 2 | 1 | MID | 材料ID = 1（MAT1と対応） |
| Field 3 | (blank) | TID | 温度依存関数ID（未使用） |
| Field 4 | PLASTIC | TYPE | 材料非線形タイプ = **塑性** |
| Field 5 | 10.0 | H | **硬化係数 = 10.0** [応力単位] |
| Field 6 | 4 | YF | **降伏関数 = 4 (Drucker-Prager)** |
| Field 7 | 1 | HR | 硬化則 = **1 (等方硬化)** |
| Field 8 | 0.2 | LIMIT1 | **粘着力パラメータ = 0.2** [応力単位] |
| Field 9 | 30.0 | LIMIT2 | **内部摩擦角 = 30°** [度] |

#### Drucker-Prager材料パラメータ詳細

**降伏関数 (YF=4)**:
```
f(σ) = α·σₘ + √J₂' - κ = 0
```

ここで:
- `α = (2 sin φ) / [√3(3 - sin φ)]`
- `κ = (6c cos φ) / [√3(3 - sin φ)]`
- `σₘ`: 平均応力 = (σ₁ + σ₂ + σ₃)/3
- `J₂'`: 第2偏差応力不変量
- `φ`: 内部摩擦角 = **LIMIT2** = **30°**
- `c`: 粘着力 = **LIMIT1/2** = **0.1** [応力単位]

**数値計算**:
```
φ = 30° = 0.5236 rad
sin(30°) = 0.5
cos(30°) = 0.866

α = (2 × 0.5) / [√3(3 - 0.5)]
  = 1.0 / [1.732 × 2.5]
  = 0.231

κ = (6 × 0.1 × 0.866) / [√3 × 2.5]
  = 0.520 / 4.330
  = 0.120
```

**硬化則 (HR=1)**:
- 等方硬化（Isotropic Hardening）
- 降伏面が一様に拡大
- Bauschingerゼ効果は考慮されない

**硬化係数 (H=10.0)**:
- 塑性域の応力-ひずみ曲線の接線勾配
- 単位: [応力単位]（MAT1のEと同じ単位系）
- この例では相対的に小さい硬化（E=200に対してH=10 → 硬化率5%）

**出典**:
- nastranhelp/pdf/QRG.pdf - MATS1 Bulk Data Entry
- nastranhelp/pdf/nonlinear_106_NXN.pdf - Section 6.2.2 (Drucker-Prager)

**寸法**: 1×1×1 立方体
**材料**: Drucker-Prager塑性材料（摩擦角30°、粘着力0.1）

### 荷重・境界条件のイメージ

        荷重 ↓↓ (Y負方向)

      1-------2 |
      | 7---F-|-8      ← 前面に圧縮荷重

### 期待される物理挙動

1. **弾性段階**（低荷重）:
   - 線形弾性応答（E=200, ν=0.3）
   - Y方向圧縮により、X・Z方向に膨張（ポアソン効果）

## 🔬 Drucker-Pragerモデルの理論背景

### 降伏条件式

```
f(σ) = α·σₘ + √J₂' - κ ≤ 0
```

**パラメータの物理的意味**:
- **α**: 圧力感度係数（摩擦角φに依存）
- **κ**: 粘着力係数（粘着力cに依存）
- **σₘ**: 静水圧応力（圧縮で増加）
- **√J₂'**: せん断応力の大きさ

### von Misesとの違い

| 項目 | von Mises | Drucker-Prager |
|------|-----------|----------------|
| **降伏条件** | √J₂' = k | α·σₘ + √J₂' = κ |
| **圧力依存性** | なし | **あり**（α項） |
| **適用材料** | 金属 | **土質、岩石、コンクリート** |
| **体積変化** | なし（関連流れ則） | **あり**（非関連流れ則可） |

### この解析での適用

**材料仮定**:
- 摩擦角30° → 砂質土、砂岩相当
- 粘着力0.1 → 若干の粘性あり

**期待される挙動**:
- 圧縮荷重下での降伏
- 塑性流動時の体積膨張（ダイレイタンシー）
- 等方硬化による強度増加
